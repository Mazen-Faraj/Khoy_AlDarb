⛺ Khoy AlDarb | خوي الدرب
AI-Powered Global Travel Consultant
Khoy AlDarb is an intelligent travel assistant built with Python and Generative AI. It provides expert travel recommendations, hotel suggestions, and cultural insights worldwide, delivered with a unique local Saudi persona.

🚀 Live Demo
Experience the app here: Khoy AlDarb Live App

🛠️ Features
Global Expertise: Specialized knowledge in global tourism destinations and logistics.

Cultural Identity: Features a "Saudi White Dialect" persona for a friendly, localized experience.

Dynamic UI: Smooth toggle between Dark Mode and Light Mode.

Secure Architecture: Uses Streamlit Secrets for API key protection.

💻 Tech Stack
Language: Python

Framework: Streamlit (Custom CSS3 & UI/UX)

AI Engine: Cohere API (Command-R Model)

Security: Streamlit Secrets Management

Deployment: Streamlit Cloud & GitHub

⚙️ Development Highlights
Prompt Engineering: Optimized system prompts for a consistent brand voice.

Responsive Design: Custom CSS for symmetry and RTL support.

State Handling: Efficiently managed chat history using st.session_state.
